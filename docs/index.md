## 迷ったときの判断フロー

```
表現を考えた
  ↓
「治る・完全・必ず・100%」系 → ❌ 即NG
  ↓
数値・試験結果を使う → 確定注釈3種を付ける
  ↓
まだ迷う → LEに相談（入稿2週間前まで）
```

---

!!! warning "両法令クリアが前提"
    薬機法と景表法は**両方同時にクリアする**表現を選ぶこと。  
    どちらか一方を守るために、もう一方を犠牲にする判断はNG。
