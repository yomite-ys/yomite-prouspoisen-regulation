## 基本構造

| 規制の種類 | 内容 |
|-----------|------|
| 不当表示の禁止 | 商品・サービスの内容や価格について、消費者を誤認させる表示を禁止 |
| 景品類の規制 | 過大な景品・プレゼントを禁止 |

<div class="section-label" id="yuryo">SECTION 01</div>

## 不当表示の禁止（第5条）

### 優良誤認表示（第5条第1号）

商品・サービスの**品質・規格・内容**について、実際よりも著しく優良であると誤認させる表示。

#### D2C広告でよくある違反パターン

| NG表示例 | 問題点 |
|---------|-------|
| 「〇〇が確実に痩せる」 | 効果を断言する表現（根拠なし） |
| 「No.1処方」「最高品質」 | 根拠のない最上級表現 |
| 「医師推奨」 | 推奨の事実・条件が不明瞭 |
| 「〇〇kg減量できました」 | 個人の感想を効果として訴求 |
| 「天然成分100%」 | 成分の定義が曖昧・誤解を招く |
| 「打ち消し表示」で本体を隠す | 強調表示と打ち消し表示の乖離 |

#### 合理的根拠（第7条）

消費者庁から根拠資料の提出を求められた場合、**15日以内**に提出が必要。根拠が不十分な場合、優良誤認表示とみなされる。

**合理的根拠として認められるもの：**

1. **第三者機関による試験・調査**（最も信頼性が高い）
2. **一般的な試験方法に依拠した自社データ**（当該分野で確立された方法に従うことが条件）
3. **一般的に認められている専門家の見解または学術文献**

**合理的根拠として認められないもの：**

- **体験談・口コミ**：原則として合理的根拠にならない
- **動物実験データのみ**：人（ヒト）のデータが必要
- **一方的に都合のいい箇所だけを抜粋した学術論文**

**根拠の2要件：**

- 要件A：提出資料が**客観的に実証**された内容であること
- 要件B：表示内容と提出資料が**適切に対応**していること（例：30代男性のデータで60代女性への効果を訴求するのはNG）

<div class="section-label" id="yuri">SECTION 02</div>

### 有利誤認表示（第5条第2号）

商品・サービスの**価格・取引条件**について、実際よりも著しく有利であると誤認させる表示。

#### D2C広告でよくある違反パターン

| NG表示例 | 問題点 |
|---------|-------|
| 「通常価格10,000円 → 今だけ3,000円」 | 通常価格が実際に販売されていない架空の価格 |
| 「定価より〇〇%OFF」 | 比較対象の価格が不明確・不正確 |
| 「今だけ！期間限定」の常時表示 | 実際には常に同価格で販売している |
| 「先着100名様」の不実表示 | 実際には制限がない |
| 「初回無料」で実は条件あり | 無料条件が不明瞭 |

#### 二重価格表示のルール

- 比較対象の価格は**同一商品を直近8週間のうち過半数の期間で販売していた価格**であること
- 販売実績のない「参考価格」「希望小売価格」をむやみに使うのはNG
- セール価格を設定する場合、元の価格で一定期間実際に販売していることが必要

<div class="section-label" id="打ち消し">SECTION 03</div>

## 打ち消し表示のルール

### NGになるケース

- 強調表示（大きな文字・目立つ色）と打ち消し表示（小さな文字・薄い色）が乖離している
- 打ち消し表示がスクロールしないと見えない位置にある
- 打ち消し表示の文字が背景色と同化していて読みにくい
- 動画広告で打ち消し表示の表示時間が極端に短い

### LP・バナー制作時の注意点

- 「※個人の感想です。効果を保証するものではありません」は免罪符にならない
- 強調表示と打ち消し表示は同程度の視認性で表示すること
- 口コミ・レビューを掲載する場合、恣意的な選別や改変はNG

## 景品規制

### 景品類の種類と上限額

| 景品の種類 | 条件 | 上限額 |
|-----------|------|--------|
| 一般懸賞（抽選等） | 取引価格が5,000円未満 | 取引価格の20倍 |
| 一般懸賞（抽選等） | 取引価格が5,000円以上 | 10万円 |
| 総付景品（全員にプレゼント） | 取引価格が1,000円未満 | 200円 |
| 総付景品（全員にプレゼント） | 取引価格が1,000円以上 | 取引価格の20% |
| 共同懸賞 | — | 30万円 |

### D2Cで注意が必要な景品ケース

- 「購入者全員にプレゼント」→ 総付景品規制の対象
- 「レビュー投稿でプレゼント」→ 景品に該当する可能性あり
- 「友達紹介で〇〇円分ポイント」→ 景品規制の検討が必要
- オープン懸賞（購入不要の抽選）→ 景品規制の対象外

## ステルスマーケティング規制（2023年10月施行）

### 対象となる表示

事業者が依頼・関与した広告であるにもかかわらず、**一般消費者が広告と認識できない表示**は不当表示。

### D2Cで注意が必要なケース

| ケース | 対応 |
|-------|------|
| インフルエンサーへの商品提供・報酬あり | 「PR」「広告」「プロモーション」等の明示が必須 |
| アフィリエイト広告 | 事業者が表示内容に関与している場合は「PR」等の明示が必要 |
| 口コミサイトへの投稿依頼 | 事業者が依頼した旨の明示が必要 |
| 自社社員によるレビュー投稿 | 関係性の明示が必要 |

### 表示方法の注意点

- 「PR」「広告」「Sponsored」「プロモーション」等、消費者が広告と認識できる表示であること
- 表示位置は投稿の冒頭など、見落とされにくい場所に
- ハッシュタグ（`#PR`等）での表示も可だが、他のタグに埋もれていないこと

## 違反した場合の措置

!!! danger "課徴金は売上の3%"
    売上規模が大きいほど高額になる。中国電力16億円・大幸薬品6億円・メルセデス・ベンツ12億円が例。

| 措置 | 公表 | 内容 |
|-----|------|------|
| **行政指導** | **非公表** | 違法状態の是正要求。合理的根拠の提出要求（15日以内）はここから始まる |
| **措置命令** | **公表** | 不当表示の差止め・再発防止策の実施命令 |
| **課徴金納付命令** | 公表 | 不当表示をした期間の売上額の**3%**（優良誤認・有利誤認のみ） |

## D2C広告・LPチェックリスト

### 表示内容

- [ ] 効果・効能を断言する表現がないか（「必ず」「確実に」「〇〇できる」等）
- [ ] 根拠のないNo.1・最上級表現がないか
- [ ] 「医師推奨」「専門家監修」等の表現に根拠資料があるか
- [ ] 個人の感想を効果として誤認させる表示になっていないか
- [ ] 打ち消し表示が強調表示と同程度の視認性で表示されているか
- [ ] 打ち消し表示がスクロールしないと見えない位置にないか

### 価格表示

- [ ] 二重価格表示の比較対象が実際に販売していた価格か
- [ ] 「期間限定」「先着〇名」等の表示が実態と一致しているか
- [ ] 「初回無料」「〇〇%OFF」の条件が明確に表示されているか

### 景品・プレゼント

- [ ] 景品の上限額を超えていないか
- [ ] レビュー投稿特典が景品規制に抵触しないか

### ステマ規制

- [ ] インフルエンサー・アフィリエイト広告に「PR」等の表示があるか
- [ ] 表示位置が見落とされにくい場所にあるか
