## 通信販売の広告表示義務（第11条）

通信販売の広告には以下を必ず表示すること。

| 表示事項 | 内容 |
|---------|------|
| 販売価格 | 消費税込みの総額表示 |
| 送料 | 送料が別途発生する場合はその金額 |
| 代金の支払時期・方法 | クレカ・コンビニ払い等の方法と時期 |
| 商品の引渡時期 | 発送・提供までの目安 |
| 返品・交換の条件 | 返品可否・期間・送料負担の明示 |
| 事業者情報 | 販売業者名・住所・電話番号・代表者名 |
| 申込みの有効期限 | 期間限定の場合 |

## 定期購入の規制（2022年改正）

**申込み最終確認画面**での必須表示：

- 定期購入である旨
- 各回の分量・金額
- 支払総額
- 解約条件・方法

!!! danger "誤認させる表示はNG"
    最終確認画面で条件を隠す等はNG。  
    違反した場合、消費者に申込みの取消権が発生する。

## 返品・解約ルール

- **返品特約がない場合**：商品受取から8日以内は返品可（送料は消費者負担）
- **返品特約がある場合**：特約内容が広告に明示されていれば特約が優先
- **定期購入の解約**：消費者が解約申込みをした場合、事業者は解約を妨害してはならない

## 罰則

| 違反内容 | 処分 |
|---------|------|
| 禁止行為・書面不交付等 | 業務停止命令（最長2年） |
| 著しい違反 | 業務禁止命令（最長5年） |
| 書類不交付・不実記載 | 1年以下の懲役または100万円以下の罰金（法人は1億円以下） |
| 誇大広告 | 指示・業務停止命令 |
