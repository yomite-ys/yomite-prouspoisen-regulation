# ポイセン 広告表現チェックシート

**プルーストウルトラ消臭リキッド** ／ 雑貨（衣類用液体洗剤）

!!! info "このシートの使い方"
    広告・LP・クリエイティブを制作する際、表現テーマごとにOK/NGを確認してください。  
    **迷ったら即LEへ。** 入稿後の修正は景表法違反・ブランド毀損のリスクになります。

---

## 大原則

| | ルール |
|--|--------|
| ✅ | 試験で実証された機能のみ訴求できる（消臭・除菌・抗菌） |
| ✅ | 数値を使う場合は必ず確定注釈文をセットで使う |
| ❌ | 「治る」「なくなる」「完全」など断定・保証はすべてNG |
| ❌ | 薬機法リスクを下げるために景表法違反を許容するのはNG（両方クリアが前提） |

## ① 消臭効果の訴求

| 表現テーマ | ✅ OK | ❌ NG |
|-----------|-------|-------|
| ワキガ臭 | 「ワキガ臭※」（注釈必須） | 「ワキガ臭を消す」（断定） |
| 消臭率 | 「99.9%消臭※」（確定注釈付き） | 「完全消臭」「ニオイがなくなる」 |
| 効果の範囲 | 「ニオイの成分を除去できる」 | 「無臭になる」「ニオイそのものをなんとかできる」 |
| 加齢臭 | 記載なし（試験値73.7%のため訴求は慎重に） | 「加齢臭にも効く」（他の臭気と同列に扱うのはNG） |
| セット訴求 | 「ポイセンで洗って、プル２で予防」 | 「プル２＋ポイセン＝無臭！」（断定） |

!!! note "ワキガ臭の注釈（必須）"
    ※腋臭成分（3MSH）のこと。

## ② 除菌・抗菌効果の訴求

| 表現テーマ | ✅ OK | ❌ NG |
|-----------|-------|-------|
| 除菌率 | 「99.9%除菌※」（確定注釈付き） | 「全ての菌を除去」「完全除菌」 |
| 抗菌 | 「99.9%抗菌※」（確定注釈付き） | 「菌が増えない」（無条件断定） |
| 菌名の強調表示 | 使用不可（強調表示での菌名記載は薬機法NG） | 「黄色ブドウ球菌を除菌！」（見出しや強調に使用） |
| 殺菌 | 「作用機序として殺菌力がある」 | 「殺菌効果がある」（直接表現） |

!!! note "除菌の注釈（必須）"
    ※菌種ではなく菌数を除去することです。全ての菌を除去するわけではありません。

!!! note "抗菌の注釈（必須）"
    ※全ての菌の増殖を抑えるわけではありません。衣類において検出される代表菌1種を用いた実験環境下で、洗浄液（洗濯水）中における菌の増殖抑制効果を評価。

## 確定注釈文（コピペ用）

!!! success "99.9％消臭※"
    **注釈文：**  
    ※ニオイの種類・度合いにより消臭の程度は異なります。実験環境下で、密閉容器内における特定臭気成分（3MSH）の低減効果を評価。

!!! success "99.9％除菌※"
    **注釈文：**  
    ※菌種ではなく菌数を除去することです。全ての菌を除去するわけではありません。

!!! success "99.9％抗菌※"
    **注釈文：**  
    ※全ての菌の増殖を抑えるわけではありません。衣類において検出される代表菌1種を用いた実験環境下で、洗浄液（洗濯水）中における菌の増殖抑制効果を評価。

!!! success "「ワキガ臭※」使用時"
    **注釈文：**  
    ※腋臭成分（3MSH）のこと。

**注釈ルール**

- フォントサイズ：**8ポイント以上**
- 必ず**注釈マーク（※）**を付ける
- 複数注釈がある場合は番号で対応関係を明示

## 入稿前チェックリスト

### 表現チェック

- [ ] 「完全」「必ず」「全ての」など断定・保証表現がないか
- [ ] 消臭・除菌・抗菌の数値に確定注釈3種が付いているか
- [ ] 「ワキガ臭」に注釈が付いているか
- [ ] 口コミ・体験談に「個人の感想」明記があるか
- [ ] 他社製品との比較がないか

### 価格・オファーチェック

- [ ] 在庫・完売・カウントダウンなど根拠のない限定表現がないか
- [ ] 期間限定の場合、実際の期間が明記されているか

### 媒体別チェック

- [ ] Amazon：口コミ引用・Amazonロゴを使っていないか
- [ ] 同梱物：LEチェックを通過しているか
