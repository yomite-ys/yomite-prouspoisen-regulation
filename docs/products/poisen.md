| | ルール |

## ① 消臭効果の訴求

<div class="okng-table" markdown="1">

| 表現テーマ | <span class="badge-ok">OK</span> | <span class="badge-ng">NG</span> |
| ----- | -------------------------------- | -------------------------------- |
| ワキガ臭  | 「ワキガ臭※」（注釈必須）                    | 「ワキガ臭を消す」（断定）                    |
| 消臭率   | 「99.9%消臭※」（確定注釈付き）               | 「完全消臭」「ニオイがなくなる」                 |
| 効果の範囲 | 「ニオイの成分を除去できる」                   | 「無臭になる」「ニオイそのものをなんとかできる」         |
| 加齢臭   | 訴求は慎重に（試験値73.7%のため他の臭気と同列扱いはNG）  | 「加齢臭にも効く」（同列に並べる）                |
| セット訴求 | 「ポイセンで洗って、プル２で予防」                | 「プル２＋ポイセン＝無臭！」（断定）               |

</div>

!!! note "ワキガ臭の注釈（必須）"
    ※腋臭成分（3MSH）のこと。

!!! note "99.9%消臭の注釈（必須）"
    ※ニオイの種類・度合いにより消臭の程度は異なります。実験環境下で、密閉容器内における特定臭気成分（3MSH）の低減効果を評価。

## ② 除菌・抗菌効果の訴求

<div class="okng-table" markdown="1">

| 表現テーマ | <span class="badge-ok">OK</span> | <span class="badge-ng">NG</span> |
|-----------|-------|-------|
| 除菌率 | 「99.9%除菌※」（確定注釈付き） | 「全ての菌を除去」「完全除菌」 |
| 抗菌 | 「99.9%抗菌※」（確定注釈付き） | 「菌が増えない」（無条件断定） |
| 菌名の強調表示 | 注釈・試験結果の文脈内での記載 | 「黄色ブドウ球菌を除菌！」（見出しや強調に使用） |
| 殺菌 | 「作用機序として殺菌力がある」 | 「殺菌効果がある」（直接表現） |

</div>

!!! note "除菌の注釈（必須）"
    ※菌種ではなく菌数を除去することです。全ての菌を除去するわけではありません。

!!! note "抗菌の注釈（必須）"
    ※全ての菌の増殖を抑えるわけではありません。衣類において検出される代表菌1種を用いた実験環境下で、洗浄液（洗濯水）中における菌の増殖抑制効果を評価。
