## 試験結果（訴求の根拠）

### 消臭試験

30分間の試験において、悪臭成分に対し98%以上の消臭効果を実証。

| 臭気成分 | 消臭率 |
|----------|--------|
| ワキガ特有のニオイ（3MSH） | 99.0%以上 |
| 酸っぱいニオイ（酢酸） | 99.7%以上 |
| 足のニオイなど（イソ吉草酸） | 98.8%以上 |
| 汗や皮脂の酸化臭（n-酪酸） | 98.8%以上 |
| 尿臭など（アンモニア） | 98.0% |
| 加齢臭（ノネナール） | 73.7% |

### 除菌・抗菌試験（原液・10分間作用）

## 確定注釈文

以下の注釈文は承認済み。使用時は必ずセットで掲載すること。

!!! success "99.9％除菌※"
    ※菌種ではなく菌数を除去することです。全ての菌を除去するわけではありません。

!!! success "99.9％抗菌※"
    ※全ての菌の増殖を抑えるわけではありません。衣類において検出される代表菌1種を用いた実験環境下で、洗浄液（洗濯水）中における菌の増殖抑制効果を評価。

!!! success "99.9％消臭※"
    ※ニオイの種類・度合いにより消臭の程度は異なります。実験環境下で、密閉容器内における特定臭気成分（3MSH）の低減効果を評価。

!!! success "「ワキガ臭※」使用時の注釈"
    ※腋臭成分（3MSH）のこと。

**「ワキガ臭」について：** そのままでは使えないが、腋臭のエビデンス（消臭率99.9%）があるため注釈付きでOK。「ニオイそのものをなんとかできる」という見せ方にならないよう「ニオイの成分を除去できる」という表現にすること。
